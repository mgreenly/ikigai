# YouTube Thumbnail Prompts — Ikigenba

Ten image-generation prompts for thumbnails about **Ikigenba**: the governed
tool-and-record layer that lets frontier AI models run a small business. Each is
written for a 16:9 (1280×720) YouTube thumbnail, **infographic / vector style —
not photorealistic**. Bold, high-contrast, readable at small size, with a clear
text zone.

Recurring brand cues to keep consistent across the set:
- Palette: deep ink-navy background (`#0B1B2B`), warm amber/gold accent
  (`#F4B740`), one cool teal (`#2DD4BF`), off-white text (`#F7F4EC`).
- Motif: small uniform service "boxes/tiles" wired together by glowing lines
  (the event plane); a single outer box = "one appliance per customer".
- Tone: confident, technical, a little wry. Big legible headline, ≤6 words.

---

## 1. Hero — "Designing a Small-Business Super-Harness"

Flat infographic, 16:9, deep ink-navy background. Center: a clean vector
"harness" rendered as a glowing exoskeleton/rig made of connected modular tiles
that wraps around a small storefront icon (a tiny shop with an awning). Straps of
the harness are luminous amber cables snapping into labeled ports (CRM, LEDGER,
WIKI, EVENTS). Bold off-white headline upper-left: **"BUILDING THE SMALL-BUSINESS
SUPER-HARNESS."** Subtle blueprint grid in the background. Crisp vector lines, no
photorealism, no gradients-heavy 3D. High contrast, readable at thumbnail size.

## 2. The Office Metaphor — "Agents Clock In, Work Persists"

Isometric flat-design infographic of a tidy little office building rendered as
stacked translucent service tiles, each tile labeled with a domain (CRM, Ledger,
Wiki, Files, Mail). A generic faceless robot/agent figure (simple geometric, not
realistic) walks up to a glowing badge reader at the door. Amber speech-tag reads
**"BE THE WORLD, NOT THE MIND."** Headline across the top in bold:
**"AGENTS COME AND GO. THE COMPANY STAYS."** Navy + amber + teal palette, vector,
flat shadows.

## 3. Brain vs. World — The Bet

Split-screen infographic, hard vertical divider. LEFT half (dim, cracked,
labeled "THE MIND"): a glowing brain icon sitting on a melting iceberg, small
text "memory · skills · persona — commoditized." RIGHT half (bright, solid,
labeled "THE WORLD"): a sturdy vault/database stack with identity badge, audit
ledger, and event lines, on rising water. Big center headline:
**"DON'T BET ON THE BRAIN."** Flat vector, infographic icons, navy/amber/teal,
strong contrast. No realistic photos.

## 4. The Event Plane — Services That React

Top-down infographic schematic: six identical rounded service tiles arranged in a
ring, each emitting a small "fact" packet into a central glowing **OUTBOX → /FEED**
spine; arrows show one action rippling outward (a sale → a ledger entry → a
notification). Dotted SSE lines pulse teal. Minimal labels. Bold headline lower
band: **"ONE ACTION, EVERY SERVICE REACTS."** Pure vector infographic, flat,
high-contrast on ink-navy. Avoid any photorealistic rendering.

## 5. One Box Per Customer — The Appliance

Flat infographic of a single sealed server "appliance" box drawn as a clean
cube with a domain nameplate reading **`acct.ikigenba.com`**. Inside the
translucent cube: a neat grid of small service tiles + one SQLite cylinder each.
A one-line "deploy" button with a swap/rollback arrow loop beside it. Headline:
**"A WHOLE COMPANY IN ONE BOX."** Sub-tag in amber: "deployed, not assembled."
Vector, isometric, navy + gold, flat shadows, no realism.

## 6. MCP Plug-In — Bring Any Agent

Infographic of a universal **MCP port/socket** at center (stylized like a glowing
USB-meets-API connector). Three different simple robot-head icons (clearly
different "agents") queue to plug into the same socket, interchangeable. Tagline
banner: **"BRING ANY AGENT. SWAP IT ANYTIME."** Small caption: "agent-neutral by
design." Flat vector, bold outlines, ink-navy background, amber/teal accents.
Not photorealistic.

## 7. The Uniform Verb Set — Same Spine, Every Service

Infographic "spec sheet" look: a row of identical service cards, and beneath each
the SAME six lifecycle verbs as glowing chips — `serve` `version` `manifest`
`migrate` `backup` `restore`. Repetition emphasized with a subtle "×N" multiplier
icon. Headline: **"EVERY SERVICE, THE SAME SHAPE."** Clean schematic/blueprint
infographic, monospace labels, navy + gold, flat. No realistic imagery.

## 8. Governance & Trust Boundary — The Single Door

Flat infographic: one glowing **nginx gateway** drawn as a single guarded door /
checkpoint that every request must pass; an inspector stamp adds trusted headers
(`X-Owner-Email`, `X-Client-Id`) before forwarding to loopback tiles behind it.
Audit-trail receipt tape spools out the side. Headline: **"ONE DOOR. EVERY
REQUEST CHECKED."** Sub-tag: "governed from day one." Vector infographic,
high-contrast navy/amber, flat icons, no photorealism.

## 9. Systems of Record Stack — What an SMB Actually Runs On

Infographic vertical stack of labeled "record" slabs — **CRM · LEDGER · WIKI ·
FILES · MAIL · CRON** — each a flat tile with a tiny representative glyph
(contact card, double-entry T, book, folder, envelope, clock). A small agent icon
reaches in and pulls a thread of data through all of them. Headline:
**"THE RECORDS A SMARTER MODEL FINALLY USES WELL."** Flat vector, layered-stack
infographic, navy + amber + teal. Avoid realism.

## 10. The Blade Question — Internal Rule

Bold typographic infographic, minimal. Center: a stylized blade/razor icon
slicing through a row of feature tiles; tiles that fall away are dim ("crutches"),
tiles that survive glow amber ("the world"). Giant headline:
**"DOES A SMARTER MODEL MAKE THIS USELESS?"** Smaller line beneath:
"if yes, build it thin. if no, own it." Flat vector poster style, ink-navy
background, single amber accent, very high contrast for thumbnail legibility.
No photorealistic elements.

---

### Usage notes
- Keep on-thumbnail text to one short headline; the longer lines above are
  optional sub-tags — drop them if the image gets busy.
- For a cohesive channel look, reuse prompt #1's palette/motif language in any
  prompt you regenerate.
- If a generator struggles with embedded text, render the art first and add the
  headline in an editor.
