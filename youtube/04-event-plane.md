# 04 — The Event Plane: Services That React

Top-down infographic schematic: six identical rounded service tiles arranged in a
ring, each emitting a small "fact" packet into a central glowing **OUTBOX → /FEED**
spine; arrows show one action rippling outward (a sale → a ledger entry → a
notification). Dotted SSE lines pulse teal. Minimal labels. Bold headline lower
band: **"ONE ACTION, EVERY SERVICE REACTS."** Pure vector infographic, flat,
high-contrast on ink-navy. Avoid any photorealistic rendering.

Palette: ink-navy `#0B1B2B`, amber `#F4B740`, teal `#2DD4BF`, off-white `#F7F4EC`.
